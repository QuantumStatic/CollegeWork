Information about the code is below
	1.	The source file(client.py) submitted was made & tested on Python 3.8.5 64 bit. 
	2.	To run the code, open the file in your IDE and just run it. (VS code has been used for development of this source code)
	3.	Inputs require IP address & Port# separated by a space. For e.g. “127.0.0.1 16000”
	4.	Code runs as specified in the assignment
	5.	Code spans 65 lines & 51 lines without spacers and line breaks 
	6.	Code handles any error that might be raised during runtime. Although after the code encounters a runtime it will stop executing, after printing the error. 
	7.	The regular expressions library(re) has been used identify the final message, since the final message is of the form “.\x\0\x” or similar, due to c++ transmission or utf-8 conversion issue. 
	8.	While testing, it was encountered that while doing POST command, you can send messages one by one instead of making it one bring string. Server will not only accept it but also return acknowledgement.
